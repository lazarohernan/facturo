# Onboarding Images

Este directorio contiene las imágenes para las pantallas de onboarding.

## Imágenes requeridas:

1. **onboarding_invoice.png** - Imagen para la sección de facturación
   - Tamaño recomendado: 400x300px
   - Formato: PNG con transparencia
   - Contenido: Visualización de facturas, documentos o gestión financiera

2. **onboarding_ocr.png** - Imagen para la sección de OCR
   - Tamaño recomendado: 400x300px  
   - Formato: PNG con transparencia
   - Contenido: Escaneo de documentos, cámara o reconocimiento de texto

3. **onboarding_analytics.png** - Imagen para la sección de analytics
   - Tamaño recomendado: 400x300px
   - Formato: PNG con transparencia
   - Contenido: Gráficos, estadísticas o dashboard de negocios

## Notas:

- Las imágenes deben tener fondo transparente o blanco
- Mantén el estilo visual consistente entre todas las imágenes
- Usa colores que complementen la paleta de la app (azules: #1E3A8A, #4F7AC7, #2563EB)
- El sistema mostrará un placeholder gradiente si las imágenes no están disponibles

## Placeholder:

Mientras no agregues las imágenes reales, el sistema mostrará placeholders con gradientes e iconos que coinciden con el tema de cada sección.
